## Overview

This repository contains SQL scripts developed for use with the **HOSxP hospital information system**.

The scripts are used for:
- Data extraction
- Reporting
- Operational analysis
- Supporting healthcare workflow decisions



## Contents

- Patient data extraction queries
- Visit / encounter reports
- Service utilization reports
- Custom hospital reporting scripts
- Data validation queries

## Query Categories

Organized by use case:
- Patient Demographics
- OPD  Reports
- Financial / Billing Queries
- Operational Analytics

## Environment

- HOSxP Hospital Information System
- MySQL / SQL-based database
- Hospital operational databases


## Usage

Run SQL scripts directly on HOSxP database environment.

⚠ Ensure proper permissions before executing queries in production systems.



## Disclaimer

These scripts are written for **internal healthcare data usage and reporting purposes only**.

They are not affiliated with HOSxP software.
